# Vampir Koylu

Browser-based Vampire Village / Werewolf round manager for party games.

## MVP features

- Host creates a room with vampire, villager, seer, and doctor counts.
- Friends join through a room code, link, or QR-style join card.
- Players enter a name and rectangular avatar photo before joining.
- Roles are assigned by the game, including the moderator-player.
- Voting is designed for phones and can be locked before elimination.
- Demo mode fills the room with bots so one real phone/player can test every phase.

## Development

```bash
npm install
npm run dev
```
